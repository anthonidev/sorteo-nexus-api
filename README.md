# Sorteo Nexus

1. Clonar el repositorio

```bash
git clone
```

2. Clonar el archivo .env.example y renombrarlo a .env

```bash
cp .env.example .env
```

3. Instalar las dependencias

```bash
pnpm install
```

4. Levantar el servidor

```bash
pnpm run start:dev
```
